## 🚀 本地运行

```bash
# 克隆项目
git clone https://github.com/SZMY-haruhi/my-social-app.git

# 进入目录
cd my-social-app

# 安装依赖
npm install

# 启动项目
npx expo start
```

用手机扫描二维码，通过 **Expo Go** 即可预览。
